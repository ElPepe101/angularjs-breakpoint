AngularJS Breakpoint

AngularJS directive that set breakpoints for responsive designed websites

  • Just add a directive to the body tag
  • Add as many breakpoints as you want
  • Current breakpoint and width available in scope

Demo

Resize the window to see the new breakpoints

Current Class:
{{breakpoint.class}}
Current Window Width:
{{breakpoint.windowSize}} (px)

Setup

Add the breakpoint attribute to the body tag and give it a value off an object defined as follows:
Set the key as the smallest value you want the breakpoint to start at and the value as the name of the class you want added to the body tag.
The values do not needto be in numerical order.
Recomment that the lowest value is 0

<body breakpoint="{0:'smallscreen', 750:'break750', 500:'break500', 1000:'break1000', 1500:'break1500'}">
...
</body>
